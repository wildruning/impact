Improved Prompt
# Role and Objective
Develop a Standard Operating Procedure (SOP) for commercial flat roof
installations using TPO membrane. Ensure all necessary safety protocols
are included.

# Instructions
- Provide clear, step-by-step instructions for TPO membrane installation
  on commercial flat roofs.
- Integrate relevant safety guidelines throughout the procedure.

## Sub-categories
- Materials and tools required
- Pre-installation site inspection
- Safety procedures
- Roof surface preparation
- TPO membrane installation steps
- Quality checks and cleanup

# Context
- Applicability: Commercial flat roofs using TPO membrane systems.
- Scope: SOP must be field-practical and address both process and safety.

# Planning and Verification
- Present the SOP as clear, numbered steps.
- Confirm all common safety and installation issues are covered.

# Output Format
- Use Markdown with appropriate headings, numbered/bulleted lists.
- Format examples and checklists as lists or tables as needed.

# Verbosity
- Maintain concise, clear instructions—sufficiently detailed for field crews.

# Stop Conditions
- SOP must fully address installation and safety for TPO flat roofs.
- Return only when the entire scope is completed and field-ready.
